# 🌤️ Weather Image Classifier (YOLO + Flask)

A simple end-to-end machine learning web application that classifies weather conditions from images using a YOLO-based deep learning model and a Flask backend.

---

## 🚀 Features

- Upload an image through a web interface
- Real-time weather classification (e.g., Rain, Cloudy, Shine, Sunrise)
- Displays prediction with confidence scores
- Built using a trained YOLO classification model
- Simple and interactive frontend

---

## 🧠 Tech Stack

- Python 🐍
- Flask 🌐
- Ultralytics YOLO 🧠
- HTML, CSS, JavaScript 🎨
- NumPy + PIL
